# PEBS APS AI MVP 使用说明

## 1. 用户如何开始

推荐从右侧“AI 排产智能体”的动作卡片开始：

1. 点击“开始使用”，查看需要导入的数据。
2. 如果暂时没有客户数据，点击“生成模拟数据”，下载样例 CSV 或一键载入模拟数据。
3. 如果使用模拟数据，一键载入后无需再点“应用导入”，可直接查看排产结果。
4. 如果已有客户数据，点击“导入订单”，粘贴或上传订单 CSV。
5. 点击“导入工艺路线”，粘贴或上传工艺路线 CSV。
6. 点击“导入设备”，粘贴或上传设备资源 CSV。
7. 确认字段智能匹配是否正确。
8. 点击“应用导入”，系统会刷新排产方案。
9. 在甘特图、订单风险、加工时序分组视图里查看结果。
10. 继续在智能体对话中询问延期、瓶颈、设备负荷、计划日报等问题。

右侧动作卡片会根据用户问题动态变化：询问“如何使用”时展示入门和导入动作；询问“模拟验证”时展示模拟数据、插单、停机、缺料动作；询问延期或瓶颈时展示分析和报告动作。

排产结果确认后，可以在甘特图区导出：

- 排产计划表：一个 XLSX 文件，包含“按设备 / 按订单 / 按零件”3 个工作表。
- 物料检查计划表：点击后先在页面查看齐套检查清单，再点击表格右上角导出。
- 物料配送计划表：点击后先在页面查看设备配送计划，再点击表格右上角导出。

## 2. 必须导入的文件

### 2.1 订单文件

用于告诉系统要生产什么、数量多少、什么时候交付。

必填字段：

- 订单号。
- 零件号/料号。
- 数量。
- 交期。

建议字段：

- 客户。
- 优先级。
- 订单类型。
- 状态。

### 2.2 工艺路线文件

用于告诉系统每个零件要经过哪些工序，每道工序可以在哪些设备上加工。

必填字段：

- 零件号/料号。
- 工序序号。
- 工序名称。
- 可选设备。

建议字段：

- 工序编码。
- 前置工序。
- 准备时间。
- 单件加工时间。
- 检验时间。
- 是否外协。
- 工装需求。

注意：多个可选设备用 `|` 分隔，例如 `CNC-01|CNC-02`。

### 2.3 设备资源文件

用于告诉系统有哪些设备，每台设备的编号、名称、能力和班次日历。

必填字段：

- 设备编号。
- 设备名称。
- 日历编号。

建议字段：

- 设备类型。
- 工作中心。
- 能力标签。
- 设备状态。
- 替代设备组。

## 3. 可选增强文件

以下文件不是 MVP 首次试排必需，但会显著提升排产可信度：

- 班次日历：每台设备每天可用时间、加班时间、停机时间。
- 物料齐套：物料是否可用，预计到料时间。
- 在制品：当前订单已做到哪道工序，剩余数量。
- 设备停机计划：维护、故障、保养时间。
- 工装夹具：模具、夹具、刀具、检具占用情况。

## 4. 字段智能匹配

字段智能匹配只在导入客户真实 CSV 时出现。使用“生成模拟数据”时，系统已经提供了标准字段，因此不需要再次匹配，也不需要点击“应用导入”。

客户上传的文件表头不需要完全等于系统模板。系统会自动识别常见中文、英文、ERP/MES 表头，并映射到系统字段。

示例：

- `订单编号`、`客户订单号`、`so_no` 可匹配到订单号。
- `料号`、`物料编码`、`产品编码` 可匹配到零件号。
- `需求数量`、`订单数量`、`qty` 可匹配到数量。
- `交货日期`、`要求交期`、`due_date` 可匹配到交期。

如果自动匹配不准确：

- 在“字段智能匹配”区域手动选择对应字段。
- 必填字段缺失时会标红。
- 必填字段未匹配时不能生成有效排产。

## 5. 推荐使用方式

首次客户试用建议只准备三份表：

- 订单。
- 工艺路线。
- 设备资源。

先跑出基础排产，再逐步补充班次、物料和在制品。这样客户能最快看到甘特图、瓶颈设备、延期风险和加工时序。

如果客户暂时无法准备数据，可先使用“生成模拟数据”：

- 下载订单 CSV。
- 下载工艺 CSV。
- 下载设备 CSV。
- 查看格式后再准备客户真实数据。
- 或点击“一键载入模拟数据”直接体验排产流程。

插单、停机、缺料三个场景也可以使用模拟数据检测：进入场景预演后，点击“载入模拟数据并检测”，系统会先检查订单、工艺路线、设备是否能生成可排工序；检测通过后再点击“执行重排”。

## 6. 三个导入标签的作用

“订单 / 工艺路线 / 设备”三个标签只服务于真实客户文件导入：

- 订单：把客户订单表映射为系统需要的订单号、零件号、数量、交期。
- 工艺路线：把 BOM/工艺表映射为零件、工序顺序、可选设备和标准工时。
- 设备：把设备台账映射为设备编号、名称、能力标签、日历和状态。

如果客户只是体验模拟数据，这三个标签可以不用看；如果客户带着自己的 Excel/CSV 来试排，这三个标签就是把客户字段转换成本系统标准输入的入口。
